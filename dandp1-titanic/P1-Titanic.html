Create Survival Label Column¶

In [6]:
titanic['Survival'] = titanic.Survived.map({0:'Died',1:'Survived'})

Create Pclass Label Column¶

In [7]:
titanic['Class'] = titanic.Pclass.map({1 : 'First Class', 2 : 'Second Class', 3 : 'Third Class'})

Create Embarked Labels Column¶

In [8]:
titanic['Ports'] = titanic.Embarked.map({'C' : 'Cherbourg', 'Q' : 'Queenstown', 'S' : 'Southampton'})

Create Person Column including Male, Female, Child (Age < 16)¶

In [9]:
def male_female_child(passenger):
    age, sex = passenger
    if age < 16:
        return 'child'
    else:
        return sex
In [10]:
titanic['Person'] = titanic[['Age','Sex']].apply(male_female_child, axis = 1)

Create Family Column¶

In [11]:
def have_family(passenger):
    SibSp, Parch = passenger
    if SibSp > 0 or Parch > 0:
        return 'Yes'
    else:
        return 'No'
In [12]:
titanic['Have_family'] = titanic[['SibSp','Parch']].apply(have_family,axis = 1)

Modify Cabin Column¶

In [13]:
# Add Cabin_class column which extracts the first letter of cabin

# Add Have_cabin binary column

In [15]:
# Change NaN value to 'None'
titanic.loc[titanic['Cabin_class'].isnull(), 'Cabin_class'] = 'None'

Conclusion:¶

  • Being female, being child, in higher class, having a cabin are positive correlated with survival.
  • The factor being female contribute most to survival.

However, there are some caveats and limitations.

  • There are clearly some multi-correlation between features such as class and cabin. One variable could be explained by another variable. Having a cabin mostly happens in first class.
  • Correlation does not imply causation. There are might some other features which influence survival rate, such as being crew or not.
  • There are many NA for example in age feature. I imputed NA with median age value. A better way should be considered, such as KNN.

Nevertheless, being female is the most powerful predictor of survival.
